1. Buttons

1.1 Usage scenario example

"Create account" button is Primary. After click there is a Secondary "Continue" button leading to the next step in which you can go Back (Third), or Forward (Secondary). In the very final step there is a Call to Action "Finish" button.

Primary
Navigating to the new page/category.
Secondary
Common navigation or/and show more purposes.
Finish
Finishing action.
Tertiary
Navigation purposes, mostly for going back.
